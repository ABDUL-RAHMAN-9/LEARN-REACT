## üí° Key Takeaways from this Lab

*   **Modular Component Thinking**: Shifted from monolithic pages to a **Component-Based Architecture**, breaking complex UIs into small, testable, and highly reusable pieces.
*   **Declarative UI Paradigm**: Mastered the React philosophy of describing *what* the UI should look like based on current `state`, letting React handle the heavy lifting of DOM manipulation.
*   **Utility-First Productivity**: Leveraged **Tailwind CSS** to eliminate the overhead of traditional CSS files, enabling rapid prototyping and ensuring consistent design patterns through utility classes.
*   **State Management Patterns**: Gained deep experience in managing data flow, from simple parent-child props to complex asynchronous state handling.

## üöÄ How to Run These Projects

Each folder in this repository is a standalone React application. To explore a project locally, follow these steps:

1.  **Navigate** into the specific project directory:
    ```bash
    cd [Folder_Name]
    ```

2.  **Install** the necessary dependencies:
    ```bash
    npm install
    ```

3.  **Launch** the development server:
    ```bash
    # For projects built with Create React App (CRA):
    npm start

    # For projects built with Vite:
    npm run dev
    ```

> _Note: Ensure you have **Node.js** installed on your machine before running these commands._
